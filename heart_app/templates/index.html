About CardioSense

End-to-end heart disease prediction powered by four machine learning algorithms.

Logistic Regression
Linear binary classifier using a sigmoid function. Highly interpretable, ideal as a medical baseline. Outputs calibrated probability estimates.
Random Forest
Ensemble of 100 decision trees. Handles non-linearity and interactions well. Provides native feature importance scores — highest performing model.
Support Vector Machine
Finds optimal hyperplane maximising margin between classes. Uses RBF kernel for non-linear boundaries. Robust against overfitting on small datasets.
K-Nearest Neighbours
Classifies patients based on 5 most similar training samples. No explicit training phase — all computation at inference time. Simple and intuitive.
Dataset Information
SourceKaggle Heart Disease Dataset (Cleveland UCI)
Records1,025 patients
Features13 clinical and physiological attributes
TargetBinary — Heart Disease Present (1) / Absent (0)
Split80% Training / 20% Testing (Stratified)
Validation5-Fold Stratified Cross-Validation
BackendPython · Flask · scikit-learn
